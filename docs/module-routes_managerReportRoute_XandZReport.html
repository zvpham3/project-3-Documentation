Module: routes/managerReportRoute/XandZReport

X and Z Report routes for manager analytics. Handles cash register reporting for shift management and end-of-day reconciliation. - X Report: Mid-shift report showing hourly sales since last Z Report without clearing totals - Z Report: End-of-day report that displays sales and sets a new baseline timestamp Uses cookies to track the last Z Report time for accurate reporting windows.

(inner) SetZReportTime

GET /manager/managerReports/xAndzReport/api/set-z-report-time Finalizes a Z Report by setting the current timestamp as the new baseline. Sets a long-lived cookie to track when the last Z Report was completed. This creates a new reporting window for subsequent X and Z Reports. After setting the timestamp, redirects back to the Z Report view.

(async, inner) generateXReport(latestZReport) → {Promise.<Array.<Object>>|string|number|number}

Generates an X or Z Report showing hourly sales breakdown. Analyzes orders since the last Z Report timestamp (or since start of current day if no Z Report exists). Groups sales by hour and includes a TOTAL row summarizing all sales in the period.
