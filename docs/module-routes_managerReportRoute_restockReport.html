Module: routes/managerReportRoute/restockReport

Restock report route for manager inventory management. Generates reports identifying inventory items below minimum threshold levels and provides automated restocking functionality to replenish inventory to maximum levels. Integrates with the restock table to track min/max quantities and shelf life.

(inner) GetRestockReport

GET /manager/managerReports/restock Generates and displays a restock report showing all items needing replenishment. Identifies inventory items at or below their minimum threshold levels.

(inner) RestockItems

GET /manager/managerReports/restock/api/restock-items Executes automated restocking for all items below minimum threshold. Replenishes each low-stock item to its maximum quantity level. Processes all items in sequence, logging successes and failures individually.
